[size=5][b]Description[/b][/size]

Tired of losing track of time while exploring the dangerous lands of Valheim? **Valheim Clock** adds a clean, unobtrusive analog clock to your HUD that displays the current in-game time. Whether you're planning your adventures around the day/night cycle or just want to know when to head back home before dark, this mod has you covered.

The clock seamlessly integrates into the game's UI, appearing only when you're actually playing - no clutter in menus or during loading screens.

[size=4][b]Installation Steps:[/b][/size]
[list=1]
[*]Install [url=https://www.nexusmods.com/valheim/mods/1]BepInEx[/url] if you haven't already
[*]Download Valheim Clock from the Files tab
[*]Extract the downloaded ZIP file
[*]Copy [b]ValheimPlugin.dll[/b] and [b]clock.png[/b] to your [b]BepInEx/plugins/[/b] folder
[*]Launch Valheim and enjoy!

[b]Typical installation path:[/b]
[code]C:\Program Files (x86)\Steam\steamapps\common\Valheim\BepInEx\plugins\[/code]

---

[size=5][b]Configuration[/b][/size]

After launching the game once, a configuration file will be created at:
[code]BepInEx/config/com.killerbob.valheimclock.cfg[/code]

[b]Available Settings:[/b]
[list]
[*][b]Enabled[/b] (true/false) - Toggle the clock on or off
[*][b]ClockSize[/b] (default: 100) - Adjust the size in pixels
[/list]

To edit, open the config file in any text editor, make your changes, save, and restart the game.

[size=5][b]Troubleshooting[/b][/size]

[b]Clock doesn't appear:[/b]
[list]
[*]Verify both [b]ValheimPlugin.dll[/b] and [b]clock.png[/b] are in the plugins folder
[*]Check that BepInEx is properly installed (you should see a console window when launching)
[*]Press [b]F5[/b] in-game to open the BepInEx console and check for errors
[*]Review the log file at [code]BepInEx/LogOutput.log[/code]
[/list]

[b]Clock is too big/small:[/b]
[list]
[*]Open [code]BepInEx/config/com.killerbob.valheimclock.cfg[/code]
[*]Change the [b]ClockSize[/b] value (try 75 for smaller, 150 for larger)
[*]Save and restart the game
[/list]

[b]Clock appears in menus:[/b]
[list]
[*]This shouldn't happen - please report with your log file
[/list]

[size=5][b]FAQ[/b][/size]

[b]Q: Does this affect game performance?[/b]
A: No, the mod is extremely lightweight and has negligible performance impact.

[b]Q: Will this work in multiplayer?[/b]
A: Yes! It's client-side only, so only you see your clock. Other players don't need the mod.

[b]Q: Can I change the clock face design?[/b]
A: Yes! Replace the [b]clock.png[/b] file in the plugins folder with your own 100x100 PNG image.

[b]Q: Does this work with Valheim Plus?[/b]
A: Absolutely! It's compatible with Valheim Plus and most other mods.

[b]Q: Is this safe to add to an existing save?[/b]
A: Yes, it's completely safe and won't affect your save files in any way.

[b]Q: Can I move the clock position?[/b]
A: Currently it's fixed at the top center. More positioning options may come in future updates!
